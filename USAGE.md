# 办公用品采购追踪系统 - 使用说明

## 快速启动

```bash
./start.sh
```

或者：

```bash
source venv/bin/activate
uvicorn main:app --host 0.0.0.0 --port 8000
```

然后访问：http://localhost:8000

## 功能说明

### 1. 上传文件识别

支持上传 **PDF 格式**的领用单，系统会自动识别：
- 流水号
- 申领部门
- 经办人
- 申领日期
- 物品名称、数量、购买链接

**注意**：PDF 格式识别准确率最高，推荐使用。

### 2. 手工录入

在左侧"快速添加记录"区域填写：
- 流水号（可选）
- 申领部门
- 经办人
- 申领日期
- 物品名称（必填）
- 数量
- 单价（可选）
- 购买链接（可选）

### 3. 在线编辑

右侧表格支持**直接点击编辑**所有字段：

| 可编辑字段 | 说明 |
|-----------|------|
| 物品名称 | 点击输入框直接修改 |
| 数量 | 点击数字输入框修改 |
| 单价 | 点击数字输入框修改（新增） |
| 购买链接 | 点击输入框修改 |
| 状态 | 下拉选择（待采购/已采购/已到货/已发放） |
| 发票 | 点击按钮切换（已开/未开） |
| 付款状态 | 下拉选择（未付款/已付款/已报销） |

**修改后自动保存**：光标离开输入框后自动保存到数据库。

### 4. 删除记录

点击表格右侧的"删除"按钮即可删除该记录。

## 识别优化

### PDF 识别效果 ✅

- 部门名称：正确处理断行（如"综合管理部（董事会办公室）"）
- 物品名称：自动清理单位标注（如"8节/板（单位：板）" → "8节/板"）
- 数量提取：从表格列正确提取，不会把型号当数量
- 链接处理：自动拼接商品ID（cemall链接）

### 图片识别 ⚠️

- 支持截图识别，但准确率较低
- 推荐只截取表格部分，避免UI元素
- 建议使用PDF格式以获得最佳效果

## 数据管理

### 统计信息

首页顶部显示：
- 总记录数
- 待采购数量
- 已开发票数量
- 已报销数量

### 筛选功能

右上角状态下拉框可按状态筛选记录。

## 常见问题

**Q: 识别错误怎么办？**
A: 直接在表格中点击修改即可，系统会自动保存。

**Q: 如何批量导入？**
A: 目前支持单个PDF上传，批量导入可上传多个PDF文件。

**Q: 数据会丢失吗？**
A: 数据保存在 `office_supplies.db` 文件中，关闭服务器不会丢失。

**Q: 如何备份数据？**
A: 复制 `office_supplies.db` 文件即可备份所有数据。
