# 使用说明

本文档对应当前版本 `1.1.1`，用于说明页面功能与操作流程。

## 1. 启动方式

### 1.1 Web 模式（开发/局域网）

```bash
./start.sh
```

打开：`http://localhost:8000`

启动时会自动执行数据库迁移（`alembic upgrade head`），用于版本平滑升级。

### 1.2 桌面模式（本机）

```bash
source venv/bin/activate
python desktop.py
```

### 1.3 Windows 一键启动

双击项目根目录：`start_windows.bat`

首次会自动创建 `venv` 并安装依赖；后续会直接启动桌面窗口。

## 2. 页面导航

左侧主导航包含 6 个页面：

- `概览`
- `台账`
- `执行看板`
- `统计报表`
- `审计日志`
- `系统设置`

说明：
- 顶部已简化，不再放置全局运维按钮
- `备份/恢复/WebDAV/回收站/数据巡检` 已迁移到 `系统设置`

## 3. 概览页（Dashboard）

- 显示核心指标：总记录数、待采购、发票、报销等
- 提供快捷操作：
  - `上传采购单据`
  - `手动新增`
  - `导出 EXCEL`
  - 跳转到执行看板/台账

## 4. 台账页（Ledger）

### 4.1 查询与筛选

支持组合筛选：

- 关键词（流水号 / 部门 / 物品名称）
- 采购状态
- 部门
- 月份（`YYYY-MM`）

### 4.2 在线编辑

表格中可直接编辑并提交：

- 流水号、部门、经办人、物品名称、数量、单价
- 状态、到货日期、分发日期、签收备注
- 发票状态、付款状态

### 4.3 批量操作

勾选多条记录后可：

- 批量修改（状态或付款状态）
- 批量删除（进入回收站）

### 4.4 分页

支持：

- 上一页/下一页
- 每页 20/50/100
- 页码跳转

## 5. 导入流程

### 5.1 上传解析

- 点击 `导入单据` 或概览页 `上传采购单据`
- 支持：`PDF / PNG / JPG / JPEG / JFIF`
- 上传后立即创建后台任务（返回 `task_id`）
- 前端每 2 秒轮询任务状态（`pending/processing/completed/failed`）
- 任务完成后自动打开“导入预览”窗口

### 5.2 预览校正

导入前可修改：

- 头部字段：流水号、部门、经办人、申领日期
- 明细字段：物品名称、数量、采购链接

### 5.3 重复处理

若检测到重复键 `(流水号 + 物品名称 + 经办人)`，可选择：

- `跳过重复项`
- `合并数量`
- `新增非重复项（跳过重复）`

## 6. 执行看板（Execution）

看板固定四列同屏：

- 待采购
- 已下单
- 待到货
- 待分发

操作方式：

- 支持拖拽卡片跨列流转
- 支持“一键流转”按钮
- 在待到货/待分发列可填写日期与签收备注
- 完成后流转为 `已分发`（闭环状态）

## 7. 统计报表（Reports）

点击 `刷新数据` 可重新拉取统计。
当台账页筛选条件变化后，切换到报表页会自动按最新筛选重算数据。

### 7.1 金额统计

- 当前费用总计
- 已计价金额
- 单价缺失笔数
- 部门支出占比（Top 10）
- 状态金额占比
- 月度金额趋势（最近 12 个月）

### 7.2 执行分析

- 执行漏斗（按状态记录数）
- 采购周期分布：
  - 申领到到货
  - 到货到分发
- 月度金额结构：
  - 总额 / 已付款 / 未付款 / 其他

## 8. 审计日志（Audit）

- 记录新增/更新/删除行为
- 支持按关键词、动作类型、月份筛选
- 支持对 `update/delete` 记录执行回滚（按历史版本）

## 9. 数据治理与安全操作

### 9.1 回收站

- 删除记录先进入回收站（软删除）
- 可恢复，或彻底删除

### 9.2 数据巡检

- 扫描质量问题与重复键组
- 展示问题码汇总与问题明细

### 9.3 本地备份/恢复

- `备份`：下载 zip（数据库 + 上传文件）
- `恢复`：上传 zip 并恢复（恢复前自动健康检查）

## 10. 系统设置（AI 引擎与运维）

入口：左侧导航 `系统设置`

### 10.1 AI 视觉解析引擎

- `local`：本地基础 OCR（离线）
- `cloud`：云端视觉大模型（多协议）

`cloud` 协议支持：

- `OpenAI 兼容`
- `Anthropic`
- `Google 直连`

可配置参数：

- API Key
- 模型名称（可留空使用默认模型）
- Base URL（可选，用于中转站）

## 11. WebDAV 云同步

入口：`系统设置` 页面内 `WebDAV 云同步`

可执行：

- 保存 WebDAV 配置（地址、账号、密码、远端目录、保留数）
- 测试连接
- 立即上传当前备份
- 查看远端备份并直接恢复

## 12. 动效说明

系统使用轻量动画提升可读性：

- 视图切换入场
- 报表图形级联展示
- 弹窗与提示过渡

若系统启用了“减少动态效果”（`prefers-reduced-motion`），动画会自动降级。

## 13. 常见问题

### 12.1 解析效果不稳定

- 优先使用可复制文本的 PDF
- 图片尽量保持清晰、裁掉无关区域
- 导入前务必在预览页校正关键字段

### 12.2 恢复后接口返回 503

系统恢复期间会短暂进入维护模式，请稍后重试。

### 12.3 Windows 打包报错

建议顺序：

1. `powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\setup_windows_env.ps1`
2. `powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\build_windows.ps1`
3. `powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\build_windows_installer.ps1`

### 12.4 桌面版离线白屏

当前前端默认通过 CDN 加载 `Vue/Tailwind/Axios`，完全离线环境可能出现白屏。建议首次运行保持网络可用；若要彻底离线，需要将这些前端依赖改为本地静态文件。

## 14. 数据落盘位置

默认位于运行目录：

- `office_supplies.db`
- `uploads/`
- `.webdav_config.json`
- `.gemini_config.json`

建议定期执行本地备份或 WebDAV 同步。
